# Business Model

200Swap is the Clanker Cashback Machine, a swap layer built on top of Clanker and other tokens within the Monad Ecosystem.&#x20;

Every time users make a swap through 200Swap, they earn **Flux**, a reward unit that fuels our innovative cashback system.&#x20;

200Swap turns regular token swaps into a continuously rewarding experience for the entire Monad community.
